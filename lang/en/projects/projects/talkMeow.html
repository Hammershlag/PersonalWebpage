TalkMeow: A Real-Time Chat App for Cat Lovers

Introduction

TalkMeow is a real-time chat web application created for cat lovers. The platform allows users to connect with friends, chat in real-time, and enjoy a safe and pleasant user experience. The application offers roles such as guest users, signed-in users, managers, and admins, with each role having distinct functionalities and privileges.

Guest User Functionality

  • Simplified Navigation Bar: Includes options for Home, Login, and Registration.
  • Home Page: Provides login and registration buttons for easy account access.
  • Login and Registration Pages: Allows users to log in or create new accounts.
  • Password Reset: A 3-step process for users to recover their accounts.

Logged-in User Functionality

  • Expanded Navigation Bar: Includes search functionality and user profile access.
  • Profile Management: Users can view and edit their username, password, and avatar.
  • Conversations: Real-time chat with message notifications and status indicators (sent/seen).
  • Search: Users can search for others by username and start conversations.

Manager Functionality

  • Add New Avatars: Managers can upload new avatars and manage existing ones.
  • Avatar Review: Managers can ensure avatar quality through a dedicated interface.

Admin Functionality

  • Access Control: Admins have access to all user and manager functionalities.
  • User Management: Admins can ban or unban users and manage inappropriate messages.
  • Suspicious Messages: Admins can approve or reject flagged messages and review banned users.

Key Features

  • Real-Time Messaging: Send and receive messages instantly with WebSocket support.
  • User Management: Role-based access control for guest users, regular users, managers, and admins.
  • Avatar Upload: Allows managers to upload avatars and review them for quality.
  • Security Features: Secure login, Spring Boot Security integration, and data encryption.
  • Database Handling: PostgreSQL database with Docker failover configuration.

Challenges and Learning

Developing TalkMeow was a valuable learning experience that involved implementing real-time messaging using WebSockets, role-based access control, and integrating robust security features. Technologies such as Java, Spring Boot, Thymeleaf, and PostgreSQL were crucial in building this project. The application emphasizes user safety, scalability, and seamless chat experience.
